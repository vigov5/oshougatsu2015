Đề bài mẫu

Viết chương trình tính tổng hai số.

Input
Một dòng duy nhất chứa hai số a và b, phân cách nhau bởi dấu cách.
Output
In ra một số là tổng của a và b.

Một vài test case mẫu

input
4 6
output
10
input
10 -10
output
0

Ví dụ về code trên từng ngôn ngữ

Hãy đọc kỹ để biết cách nhập input và cách in ra output tiêu chuẩn cho từng ngôn ngữ. Output chỉ được chấp nhận khi nó hoàn toàn giống output của các test cases.

Java

Note: Please set your class name to Main 

import java.util.Scanner;

public class Main {
  public static void main(String args[]){
    Scanner sc = new Scanner(System.in);
    int a = sc.nextInt();
    int b = sc.nextInt();

    System.out.print(a + b);
  }
}

  input: process.stdin,
  output: process.stdout,
  terminal: false
});

rl.on('line', function(line){
    var nums = line.split(" ");
    var a = parseInt(nums[0]);
    var b = parseInt(nums[1]);
    console.log(a +b);

    process.exit();
})
