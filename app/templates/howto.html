Example submisssions

Please see below examples to understanding how to read from standard input and write to standard output.
Your output must be strictly the same as the output of testcases.

C++

C

#include <stdio.h>
#include <stdlib.h>

  int a;
  int b;
  scanf("%d", &a);
  scanf("%d", &b);

  printf("%d", a + b);

  return 0;
}

Java

Note: Please set your class name to Main 

import java.util.Scanner;

public class Main {
  public static void main(String args[]){
    Scanner sc = new Scanner(System.in);
    int a = sc.nextInt();
    int b = sc.nextInt();

    System.out.print(a + b);
  }
}

echo $a + $b;
?>

JavaScript (NodeJS)

var readline = require('readline');
var rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
  terminal: false
});

rl.on('line', function(line){
    var nums = line.split(" ");
    var a = parseInt(nums[0]);
    var b = parseInt(nums[1]);
    console.log(a +b);

    process.exit();
})
